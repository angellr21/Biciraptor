# 🚴 Biciraptor Website

Proyecto basado en la plantilla **Xtreme Sports Club (HTML Template)** adaptado para **Biciraptor**, tienda especializada en bicicletas, repuestos, accesorios y servicios técnicos.

## 🌐 Estructura del sitio
- Inicio: index.html (landing con efecto parallax)
- Tienda: shop.html
- Producto individual: product.html
- Nosotros / Taller / Contacto: page.html
- Blog: blog.html y post.html
